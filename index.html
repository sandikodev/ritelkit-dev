Toolchain

Spesifikasi Perangkat

Setiap engineer wajib menggunakan stack standar untuk menjamin konsistensi build dan environment.

  • • Node.js v20+ (LTS)
  • • pnpm v9.x (Package Manager)
  • • Turso CLI (LibSQL Manager)
  • • Cloudflare Wrangler v3+
  • • Git v2.40+
Environment

Kebutuhan Sistem

Spesifikasi minimum untuk menjalankan seluruh ekosistem (Dashboard + App + DB) secara lokal.

  • • OS: Linux (Ubuntu/Pop!_OS) / macOS
  • • RAM: Minimal 8GB (Recommended 16GB)
  • • Local Domain Mappings (/etc/hosts)
  • • VS Code + Svelte/Astro Extensions

Development Onboarding

Langkah-langkah bagi kontributor internal untuk memulai pengembangan.

1. Akses & Clone

Dapatkan akses SSH ke tim SandiKodev, lalu jalankan clone rekursif:
git clone --recursive [URL_REPO]

2. Konfigurasi Host

Petakan domain lokal di mesin Anda:
127.0.0.1 ritelkit.local

3. Instant Runner

Gunakan script instan kami untuk menyalakan seluruh stack dev:
./dev.sh
