Developer Onboarding

Mulai pengembangan dalam hitungan menit dengan toolchain standar kami.

01

Akses & Clone

Dapatkan akses SSH tim, lalu lakukan clone rekursif untuk menyertakan seluruh submodules.

git clone --recursive [URL]
02

Domain Stable

Lupakan port. RitelKit menggunakan .localhost:1355 yang stabil untuk Dashboard dan App.

ritelkit.localhost:1355
03

Portless Runner

Jalankan runner pintar kami yang otomatis mengelola orkestrasi domain lokal.

./dev.sh

System Stack

Spesifikasi lingkungan minimal untuk memastikan kompatibilitas build antar engineer.

  • • Node.js: v20.x (LTS)
  • • Package Manager: pnpm v9+
  • • Database: Turso (LibSQL)
  • • Edge: Cloudflare Wrangler

Spec Mesin

Rekomendasi perangkat untuk pengalaman pengembangan yang paling mulus dan responsif.

  • • OS: Linux (Ubuntu/Pop!_OS) / macOS
  • • RAM: Minimal 8GB (Rec. 16GB)
  • • Editor: VS Code (Astro/Svelte Plugins)
